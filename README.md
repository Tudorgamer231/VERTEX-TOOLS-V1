## 🔥 Discord RAID Tools (Option 3)

1. Get a **bot token** from [Discord Developer Portal](https://discord.com/developers/applications)  
   - Enable **Server Members Intent** and **Message Content Intent**  
   - Invite bot with **Administrator** permission
2. In the tool, paste token & server ID (right-click server → Copy ID)
3. Choose action:

| # | Action | What it does |
|---|--------|---------------|
| 1 | Full Nuke | Delete all channels → create spam channels → ban all → rename server |
| 2 | Spam | Flood a channel with messages |
| 3 | Delete all channels | Removes every channel |
| 4 | Create mass channels | Makes many empty channels |
| 5 | Rename server | Changes server name |
| 6 | Single DM | Message one user by ID |
| 7 | Mass DM | Send to many users (paste IDs or use file) |
| 8 | Ban all members | Kicks everyone except bot |
| 9 | Edit raid config | Change message, channel count, delays |

> **Destructive actions need confirmation** – type `NUKE` or `BAN`.
